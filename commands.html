General Commands

  • !b help – Show all available commands.

Countdown Commands

  • !b c [seconds] – Starts a countdown.
  • !b playc – Resumes the countdown.
  • !b stopc – Stops the countdown.

Task Commands

  • !b addtask [task] – Add a task.
  • !b removetask [#] – Remove a task.
  • !b done [#] – Mark as done.
  • !b tasks – Show list.
  • !b cleartasks – Clear list.

Fun Commands

  • !b inspireme – Get a motivational quote.
